Project Summary

For Galen is a Next.js web application deployed on Vercel, designed as a personalized training hub for Galen, an IT Manager. The platform covers Claude Code automation, project management, Excel, Salesforce, Fishbowl, and document management through curated courses authored by distinct AI expert personas.

Training Hub

Curated MDX-based learning tracks with custom components (Callout, Quiz, StepByStep). Lessons are authored by AI personas with unique teaching styles.

AI Assistant

Context-aware streaming chat powered by Claude Sonnet. Six domain contexts (Excel, Salesforce, Fishbowl, Documents, PM, General) with persona-injected prompts.

Skills System

Interactive Skill Tutor (5-step assessment wizard) and Project Coach (Marcus Rivera PM chat). Personalized learning path generation.

Custom Courses

Galen can create his own training materials with a Markdown editor, tag management, and full CRUD support. Courses include interactive lesson chat.

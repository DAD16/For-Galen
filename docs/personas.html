System Prompt Strategy

The AI system prompt is built in layers, combining a base prompt, context-specific knowledge, and persona teaching style.

Final System Prompt = BASE_SYSTEM_PROMPT + CONTEXT_PROMPTS[selectedContext] // Domain-specific focus + persona.systemPromptAddition // Teaching style & personality Example for Excel context: ──────────────────────────────────── BASE: "You are a helpful AI assistant for Galen, an IT Manager..." + EXCEL CONTEXT: "Focus on formulas, VBA, data manipulation..." + JAMES OKAFOR: "Your teaching style is visual and example-driven..."
